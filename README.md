# Auto Video Agent

自动全流程短视频制作发布Agent

## 核心能力

1. **调研引擎** — 爬取热点/专项话题，分析趋势，输出选题池
2. **选题决策** — AI评估选题价值（流量/竞争/可行性），自动选出最优
3. **脚本写作** — 根据选题生成分段式口播脚本
4. **分镜规划** — 脚本→分镜描述（画面/时长/转场/字幕）
5. **数字人生成** — TTS语音 + HeyGen/D-ID 数字人视频
6. **素材收集/生成** — 图片/视频片段/BGM/音效
7. **剪辑合成** — FFmpeg编排，合成最终视频
8. **多平台发布** — 抖音/快手/小红书/B站/视频号
9. **编排引擎** — 全流程调度，状态机驱动，失败重试

## MVP 范围（Week 1）

调研 → 选题 → 脚本 → 数字人 → 抖音发布

## 技术栈

- Python 3.13
- HeyGen/D-ID API（数字人）
- FFmpeg（视频合成）
- 抖音开放平台 API（发布）
- SQLite（状态管理）

## 环境

- Windows 11（无GPU）
- 纯API方案（数字人/TTS/素材）
